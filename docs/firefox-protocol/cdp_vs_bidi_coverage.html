CDP vs BiDi 覆盖对比表

完整覆盖 部分覆盖 无覆盖 爬虫不相关
CDP 域 BiDi 覆盖 爬虫相关 说明 / BiDi 对应
核心控制
Network 完整 ★ 核心 network 模块：addIntercept / continueRequest / continueResponse / provideResponse / setExtraHeaders，事件覆盖 beforeRequestSent / responseCompleted 等
Fetch 完整 ★ 核心 BiDi network 模块已统一 Fetch 能力，拦截+修改请求/响应均支持
Page 完整 ★ 核心 browsingContext 模块：navigate / reload / captureScreenshot / setViewport / setBypassCSP；script.addPreloadScript 对应 addScriptToEvaluateOnNewDocument
Runtime 完整 ★ 核心 script 模块：evaluate / callFunction / addPreloadScript / getRealms，支持跨 realm 执行
Target 完整 ★ 核心 browsingContext.create/close/getTree + browser.createUserContext，多标签/多上下文管理完整
Storage 完整 ★ 核心 storage 模块：getCookies / setCookie / deleteCookies，支持 partition key
Input 完整 重要 input 模块：performActions / releaseActions / setFiles，鼠标键盘触控均支持
Emulation 完整 重要 emulation 模块：setUserAgentOverride / setGeolocationOverride / setTimezoneOverride / setViewport / setNetworkConditions 等
DOM 部分 重要 browsingContext.locateNodes 提供基础节点定位；复杂 DOM 操作（getBoxModel / resolveNode / requestChildNodes 等）需通过 script.evaluate 间接实现
Log 完整 一般 log 模块：log.entryAdded 事件，捕获 console 输出
Console 完整 一般 同上，log.entryAdded 统一覆盖
Browser 完整 一般 browser 模块：getClientWindows / getUserContexts / setClientWindowState / setDownloadBehavior
调试 / 性能分析
Debugger 部分 不相关 script.realmCreated/Destroyed 提供 realm 生命周期；断点/单步调试无 BiDi 对应（调试器场景不在 BiDi 范围内）
Profiler 无覆盖 不相关 JS CPU profiling，BiDi 无对应
HeapProfiler 无覆盖 不相关 堆内存分析，BiDi 无对应
Performance 无覆盖 不相关 性能指标采集，BiDi 无对应
PerformanceTimeline 无覆盖 不相关 同上
Tracing 无覆盖 不相关 Chrome Trace 格式录制，BiDi 无对应
Memory 无覆盖 不相关 内存压力模拟，BiDi 无对应
页面检查 / 渲染
CSS 无覆盖 不相关 CSS 样式规则检查，BiDi 无对应；爬虫场景通过 script.evaluate 读取 computedStyle
DOMSnapshot 无覆盖 一般 整页 DOM 快照，BiDi 无直接对应；可用 script.evaluate 序列化 DOM
DOMStorage 部分 一般 Cookie 通过 storage 模块覆盖；localStorage/sessionStorage 无直接命令，需 script.evaluate 读写
DOMDebugger 无覆盖 不相关 DOM 事件断点，调试用途
Overlay 无覆盖 不相关 DevTools 高亮叠加层，调试用途
LayerTree 无覆盖 不相关 渲染层分析，调试用途
Accessibility 无覆盖 不相关 无障碍树查询，BiDi 无对应
存储 / 缓存
CacheStorage 无覆盖 一般 Service Worker Cache API 检查，BiDi 无对应
IndexedDB 无覆盖 一般 BiDi 无对应；需 script.evaluate 间接操作
FileSystem 无覆盖 不相关 Origin Private File System，BiDi 无对应
安全 / 其他
Security 无覆盖 一般 证书错误处理，BiDi 无直接对应；绕过可用 acceptInsecureCerts capability
ServiceWorker 无覆盖 一般 SW 注册/调试，BiDi 无对应
Audits 无覆盖 不相关 Lighthouse 集成，BiDi 无对应
Animation 无覆盖 不相关 CSS 动画检查，BiDi 无对应
WebAudio 无覆盖 不相关 Web Audio 图检查，BiDi 无对应
WebAuthn 无覆盖 不相关 虚拟认证器，BiDi 无对应
Media 无覆盖 不相关 媒体播放内部状态，BiDi 无对应
IO 无覆盖 不相关 流式 IO handle，BiDi 无对应
Inspector 无覆盖 不相关 连接/断开事件，BiDi 用 session 模块替代
Schema 无覆盖 不相关 协议域列举，BiDi 无需此概念
SystemInfo 无覆盖 不相关 GPU/显示信息，BiDi 无对应
Tethering 无覆盖 不相关 Android 端口转发，BiDi 无对应
Cast 无覆盖 不相关 Chromecast 控制，Chrome 独有
DeviceAccess 无覆盖 不相关 USB/蓝牙设备访问，BiDi 无对应
DeviceOrientation 无覆盖 不相关 陀螺仪模拟，BiDi 无对应（emulation 模块有 setTouchOverride）
BluetoothEmulation 无覆盖 不相关 蓝牙模拟，Chrome 实验性
SmartCardEmulation 无覆盖 不相关 智能卡模拟，Chrome 实验性
FedCm 无覆盖 不相关 联合身份，Chrome 实验性
Autofill 无覆盖 不相关 自动填充测试，Chrome 专有
BackgroundService 无覆盖 不相关 后台推送/通知记录，调试用途
Preload 无覆盖 不相关 推测预加载规则，Chrome 专有
Extensions 无覆盖 不相关 Chrome 扩展 storage，Chrome 专有（BiDi 有独立 webExtension 模块用于 Firefox 扩展）
EventBreakpoints 无覆盖 不相关 事件监听断点，调试用途
HeadlessExperimental 不适用 不相关 旧版 Headless 接口，已被 Chrome Headless 新模式取代
PWA 无覆盖 不相关 PWA 安装操作，Chrome 专有
WebMCP 无覆盖 不相关 Chrome 实验性 MCP，非常新
CrashReportContext 无覆盖 不相关 崩溃报告，Chrome 内部用途

BiDi 额外新增（CDP 无对应）：webExtension 模块（Firefox 扩展安装/卸载）、browser.createUserContext（隔离 profile）、script.Channel（页面↔控制端双向消息通道）。
